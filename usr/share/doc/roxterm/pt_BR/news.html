2014-07-31 Version 2.9.1
  • Supports launching of SSH sessions from matched URIs or hostnames, while common VCS URIs are matched for copying to the clipboard.
  • Allows Shortcuts profiles to be edited as text for GTK+ 3.10 and newer.
  • Fixed background transparency support for recent versions of VTE/GTK (compositing window manager required?).

02-06-2011 Versão 1.22.1
  • Suporte de composição melhorado.
  • Adicionado a opção --role.
  • Removida as opções --xclass e --xname. --name e/ou --class são alternativas, mas não devem ser usadas em conjunção com --separate. Use --role se você precisa de um identificador por janelas.
  • Tente evitar criar arquivos de sessão redundantes. Considere executar 'rm ~/.config/roxterm.sourceforge.net/Sessions/*' para limpar os arquivos obsoletos criados antes dessa versão.
