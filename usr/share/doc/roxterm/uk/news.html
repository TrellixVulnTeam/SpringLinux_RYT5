Новини

2014-07-31 Version 2.9.1
  • Supports launching of SSH sessions from matched URIs or hostnames, while common VCS URIs are matched for copying to the clipboard.
  • Allows Shortcuts profiles to be edited as text for GTK+ 3.10 and newer.
  • Fixed background transparency support for recent versions of VTE/GTK (compositing window manager required?).
2014-01-16 Version 2.8.1
  • Terminal background options are disabled when using a recent version of VTE which deprecates them.
2013-03-29 Version 2.7.1
  • Translations are now enabled.
  • SF project has been upgraded with new git repository.
  • New behaviour for -e (see man page).
  • Several new features and bug-fixes.
24 лютого 2012 Версія 2.5.1
  • Debian-пакунки тепер керовані через git-buildpackage.
  • Припинено підтримку autoconf.
  • Покращено опції підтвердження закриття вікна чи вкладки.
  • Додано опцію темної теми.
  • --tab використовує вікна лише поточної робочої області.
  • Згенеровану графіку включено до архівів.
  • Ubuntu PPA переміщено до https://launchpad.net/~h-realh/+archive/roxterm
24 листопада 2011 Версія 2.3.1
  • Нова, власна система збірки — «maitch».
21 серпня 2011 Версія 2.0.1
  • Типово використовується GTK+3; підтримка GTK+2 все ще наявна.
  • Додано функцію пошуку.
  • Додано підтримку файлових посилань.
  • Серйозно покращено рядок вкладок.
  • GtkBuilder використовується замість libglade.
2 червня 2011 Версія 1.22.1
  • Покращено підтримку композитингу.
  • Додано параметр --role.
  • Видалено параметри --xclass та --xname. Альтернативи, --class та --name, слід використовувати разом із --separate. Вказуйте --role, якщо потрібен ідентифікатор вікна.
  • Додано обхід створення зайвих файлів сеансу. Запустіть «rm ~/.config/roxterm.sourceforge.net/Sessions/*», щоб стерти такі файли, створені старими версіями.
12 лютого 2011 Версія 1.21.1
  • Додано підтримку параметру TERM при vte >= 0.26.
  • Command line options to set window class hints.
  • Заголовок налаштовується через профілі.
17 листопада 2010 Версія 1.20.0

2010-03-23 Version 1.18.1
  • Supports X session management.
