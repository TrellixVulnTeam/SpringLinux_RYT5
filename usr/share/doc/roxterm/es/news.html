Noticias

2014-07-31 Version 2.9.1
  • Supports launching of SSH sessions from matched URIs or hostnames, while common VCS URIs are matched for copying to the clipboard.
  • Allows Shortcuts profiles to be edited as text for GTK+ 3.10 and newer.
  • Fixed background transparency support for recent versions of VTE/GTK (compositing window manager required?).
2014-01-16 Version 2.8.1
  • Terminal background options are disabled when using a recent version of VTE which deprecates them.
2013-03-29 Version 2.7.1
  • Translations are now enabled.
  • SF project has been upgraded with new git repository.
  • New behaviour for -e (see man page).
  • Several new features and bug-fixes.
2012-02-24 Versión 2.5.1
  • El empaquetado Debian ahora se gestiona mediante git-buildpackage..
  • Eliminado el soporte de autoconf.
  • Mejoradas las opciones de aviso al cerrar ventanas/pestañas.
  • Implementada la opción de tema oscuro.
  • --tab solo reutiliza ventanas en el espacio de trabajo actual.
  • Los gráficos generados se incluyen en archivos tar.
  • El PPA de Ubuntu se ha movido a https://launchpad.net/~h-realh/+archive/roxterm
2011-11-24 Versión 2.3.1
  • Nuevo sistema de compilado a medida, maitch.
2011-08-21 Versión 2.0.1
  • Utiliza GTK+3 por defecto, el soporte de GTK+2 aun está disponible.
  • Función de búsqueda añadida.
  • Soporte para URIs de archivo añadido.
  • Muchas mejoras en la barra de pestañas.
  • Utiliza GtkBuilder en lugar de libglade.
2011-06-02 Versión 1.22.1
  • Soporte de composición mejorado.
  • Añadida la opción --role.
  • Opciones --xclass y --xname eliminadas. --name y/o --class son alternativas pero se deberían utilizar junto a --separate. Utilice --role si necesita un identificador por ventana.
  • Intenta evitar crear archivos de estado de sesión redundantes. Considere ejecutar 'rm ~/.config/roxterm.sourceforge.net/Sessions/*' para limpiar los archivos de estado creados antes de esta versión.
2011-02-12 Versión 1.21.1
  • Soporte de las opciones TERM con vte >= 0.26.
  • Opciones en línea de comandos para cambiar las etiquetas de la clase ventana.
  • El título de la ventana se puede configurar en los perfiles.

  • Cambios en el manejo de pestaña/ventana.
